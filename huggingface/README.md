# Sentinel — AI-Powered Alert Triage

AI agent that triages ops alerts (Datadog/Sentry-style) using Claude, decides
whether to auto-remediate or escalate, and exposes full reasoning traces.

**Set `ANTHROPIC_API_KEY` as a Space secret before running.**

See the [GitHub repo](https://github.com/mansimengde17/sentinel) for full docs.
